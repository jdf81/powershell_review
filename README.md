# PowerShell Path PWA

An installable, offline-friendly course for learning PowerShell.

## Features
- 10 lessons from beginner to intermediate
- Code examples and practice exercises
- Lesson quizzes
- Local progress tracking
- Responsive mobile layout
- PWA install prompt
- Offline caching after the first visit

## Run locally
A service worker requires HTTP or HTTPS. From this folder, run one of these commands:

```bash
python -m http.server 8000
```

or

```powershell
npx serve .
```

Then open `http://localhost:8000`.

## Deploy
Upload the folder to GitHub Pages, Netlify, Vercel, Azure Static Web Apps, or any HTTPS static host.

## Notes
The curriculum is educational. Test administrative commands in an authorized lab environment and review scripts before running them.
